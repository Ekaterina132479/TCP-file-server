# TCP File Server

Простой учебный **TCP**‑сервер и клиент на C++, которые позволяют передавать файлы по сети.

## Структура проекта

- `server.cpp` - код сервера.
- `client.cpp` - код клиента.
- `server_folder/Files_catalog/`- каталог с файлами, которые сервер может отдавать.
- `client_folder/Downloads/` - каталог, куда клиент сохраняет полученные файлы.

## Принцип работы

1. Сервер запускается и слушает указанный порт.
2. Клиент подключается к серверу по IP и порту.
3. Клиент запрашивает файл по имени.
4. Сервер читает файл из `Files_catalog` и отправляет его клиенту.
5. Клиент принимает данные и сохраняет файл в `Downloads`.

## Сборка

```bash
# Сборка сервера
g++ -std=c++14 server.cpp -o server -pthread

# Сборка клиента
g++ -std=c++14 client.cpp -o client
```
## Запуск

Сервера:

```bash
./server <port>
```
Клиента:
```bash
./client <hostname> <port> <filename>
```
При неверном запуске программа покажет подсказки со всеми необходимыми параметрами.

## Ограничения программы:
- Проект создан в учебных целях
- Нет шифрования и аутентификации, использовать только в доверенной сети
- Обработка ошибок и проверка входных данных упрощены
